# Core Tables Reference

## `x_society_flat` — Flat/Unit Registry

Master record for every unit in the society. All other tables have a reference field pointing here.

---

## `x_society_maintenance_bill` — Maintenance Bills

One record per flat per month. Auto-generated by the monthly scheduled job. Tracks payment status and overdue state.

---

## `x_society_service_request` — Service Requests

Unified table for complaints, facility bookings, and notices. The `request_type` choice field distinguishes the three categories. A single table intentionally — keeps reporting and ACLs consolidated.

---

## `x_society_visitor_log` — Visitor Log

Gate entry/exit log maintained by security staff. References `x_society_flat` for the visited unit.
